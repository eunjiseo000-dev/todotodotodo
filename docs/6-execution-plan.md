# ToDoToDoToDo 프로젝트 실행 계획

## 프로젝트 개요

**MVP 일정**: 3일 (0.5 + 1 + 1 + 0.2 + 0.3)
**개발자**: 1명
**기술 스택**: Node.js/Express, React 18.x, Supabase PostgreSQL, Vercel

---

## 전체 일정 타임라인

```
Day 1
├── 08:00-12:00 [설계] DB 스키마 설계 및 프로젝트 초기화
├── 13:00-17:00 [백엔드] 인증 API 개발
└── 17:00-18:00 [백엔드] 기본 할일 CRUD 시작

Day 2
├── 08:00-12:00 [백엔드] 할일 CRUD 및 상태 관리 완성
├── 13:00-17:00 [프론트엔드] 인증 UI 및 할일 대시보드
└── 17:00-18:00 [프론트엔드] 폼 및 상태 관리

Day 3
├── 08:00-10:00 [프론트엔드] 우선순위 및 휴지통 UI
```
