너는 최고의 비즈니스 도메인 전문가야. "ToDoToDoToDo"라는 이름의 사용자 인증과 함께 사용하는 ToDoList 애플리케이션을 개발하려고 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 사용자마다 할일 목록을 관리할 수 있는 애플리케이션이면 좋겠다
- 할일 마다 완료여부를 체크하는 목록 관리가 되었으면 좋겠다.
- 할일 마다 시작일과 종료일을 기입해서 관리할 수 있었으면 좋겠다.

# Team CalTalk 앱의 주요 기능 :

- 회원가입을 한 다음 로그인을 한 사용자만 애플리케이션을 사용할 수 있다.
- 로그인한 사용자는 자신이 입력한 할일 목록을 확인할 수 있다.
- 할일 목록에 새로운 할일(내용, 시작일, 종료일)을 추가할 수 있다.
- 기존 할일을 수정할 수 있다.
- 기존 할일을 삭제할 수 있다.
- 삭제된 할일은 휴지통으로 분류되고 다시 원복할 수 있어야 한다.
- 작성된 할일 목록에서 완료 여부를 체크할 수 있다.
- 완료된 할일은 완료 목록으로 분류한다.
