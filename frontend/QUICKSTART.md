# 빠른 시작 가이드

## 1. 사전 요구사항

- Node.js 18.x 이상
- npm 9.x 이상
- 실행 중인 백엔드 서버 (http://localhost:3000)

## 2. 설치

```bash
# frontend 디렉토리로 이동
cd frontend

# 의존성 설치
npm install
```

## 3. 환경 설정

`.env` 파일이 이미 생성되어 있습니다:

```env
VITE_API_BASE_URL=http://localhost:3000/api
```

백엔드 서버 주소가 다른 경우 수정하세요.

## 4. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 5. 사용 방법

### 회원가입
1. 로그인 페이지에서 "회원가입" 클릭
2. 이름, 이메일, 비밀번호 입력
3. "회원가입" 버튼 클릭
4. 자동으로 로그인되어 대시보드로 이동

### 로그인
1. 이메일과 비밀번호 입력
2. "로그인" 버튼 클릭
3. 대시보드로 이동

### 할일 관리
1. **새 할일 추가**: "+ 새 할일" 버튼 클릭
2. **할일 완료**: "완료" 버튼 클릭
3. **할일 수정**: "수정" 버튼 클릭
4. **할일 삭제**: "삭제" 버튼 클릭 (휴지통으로 이동)
5. **할일 복원**: 휴지통 탭에서 "복원" 버튼 클릭
6. **영구 삭제**: 휴지통 탭에서 "영구삭제" 버튼 클릭
7. **우선순위 변경**: 할일을 드래그하여 순서 변경

### 탭 전환
- **진행중**: 완료되지 않은 할일 목록
- **완료**: 완료된 할일 목록
- **휴지통**: 삭제된 할일 목록

## 6. 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 확인
npm run preview
```

빌드된 파일은 `dist/` 폴더에 생성됩니다.

## 7. 문제 해결

### 백엔드 연결 오류
- 백엔드 서버가 실행 중인지 확인
- `.env` 파일의 `VITE_API_BASE_URL` 확인
- 브라우저 콘솔에서 에러 메시지 확인

### 로그인/회원가입 실패
- 입력값 검증 확인
- 백엔드 API 응답 확인
- 네트워크 탭에서 요청/응답 확인

### 할일 목록이 보이지 않음
- 로그인 상태 확인
- 브라우저 콘솔에서 에러 확인
- 토큰이 localStorage에 저장되어 있는지 확인

## 8. 개발 팁

### 디버깅
- React DevTools 사용 권장
- 브라우저 콘솔 확인
- Network 탭에서 API 요청 확인

### 토큰 관리
- localStorage에 `auth_token` 키로 저장
- 개발자 도구 > Application > Local Storage에서 확인 가능

### 상태 관리
- AuthContext: 인증 상태
- TodoContext: 할일 상태
- React DevTools의 Components 탭에서 확인 가능
